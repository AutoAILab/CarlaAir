# User Control

## Description
A user should be able to control how the UGV and UAVs move in the simulation for capturing the dataset.

Possible ideas:
- Set up some type of route file, where the user can define waypoints for the UGV and UAVs to follow. Controls pitch and height if needed.
- Set up follow mode, where a UxV follows another specific UxV. There much be one UxV that has a route file
